Projeto criado para a matéria de Desenvolvimento Web da faculdade.

A proposta foi desenvolver um site utilizando HTML e CSS no Front-end, PHP no Back-end e MySQL para banco de dados.

O Back-end foi desenvolvido com o paradigma orientado a objetos e no padrão MVC.

O tema escolhido foi um Pet shop.
